singleton

  • singleton<P, R>(original: Resolver<P, R>): Resolver<P, R>
    • Defined in packages/apiiz/lib/main.ts:106

    create a singleton version of spceified resolver. make sure a dispatcher does not re-create multiple times

    ```js const myResolver = () => {...} // both instances of myResolver are different, this might cause unexpected issue if myResolver contains concurrency, caching logics const api = define({ myResolver, // this is first instance otherApi: enhance(...).with(enahncer(myResolver)) // this is second instance });

    // correct way const myResolver = singleton(() => {...}); const api = define({ myResolver: myResolver, otherApi: enhance(...).with(enahncer(myResolver)) })

    ```

    Type parameters

    • P = void

    • R = any

    Parameters

    Returns Resolver<P, R>
